กำลังเชื่อมต่อ Firebase…

🔧 ตั้งค่า Firebase ก่อนใช้งาน

Dashboard นี้ใช้ Firebase Realtime Database เพื่อเก็บคะแนนโหวตแบบ real-time
ทำตาม 4 ขั้นตอนด้านล่าง แล้วแก้ไขไฟล์ HTML นี้ด้วย config ของคุณ

1
ไปที่ console.firebase.google.com → สร้าง Project ใหม่ (ชื่ออะไรก็ได้ เช่น pea-ideas-vote) → ปิด Google Analytics
2
ใน Project → Build → Realtime Database → Create database → เลือก Region: asia-southeast1 → เริ่มด้วย Test mode (30 วัน)
3
ไปที่ Project Settings (ไอคอน ⚙️) → General → เลื่อนลงถึง Your apps → เพิ่ม Web App (</>) → Copy firebaseConfig ที่ได้
4
เปิดไฟล์ HTML นี้ด้วย text editor → หาบรรทัด const firebaseConfig = { ด้านบน → แทนที่ค่า REPLACE_... ทั้งหมดด้วยค่าจาก Console → บันทึกไฟล์
💡 หลังจากตั้งค่าแล้ว นำไฟล์ไป deploy บน GitHub Pages, Netlify Drop, หรือ Cloudflare Pages ได้เลย — ฟรีทั้งหมด
